# KE Android Player

Android TV Player for [Karaoke Eternal](https://github.com/bhj/karaoke-eternal).

## Features (planned)

- Login with admin credentials
- JWT authentication
- Socket.io connection for real-time queue updates
- Video playback with ExoPlayer
- Lyrics overlay display

## Requirements

- Android TV or Android Box
- Android 7.0+ (API 24)
- Karaoke Eternal server running on network

## Development

Built with:
- Kotlin
- Jetpack Compose for TV
- ExoPlayer for video playback
- Socket.io client for real-time communication

## Setup

1. Open project in Android Studio
2. Configure server IP in app settings
3. Build and deploy to Android TV device
